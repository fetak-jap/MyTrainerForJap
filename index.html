ひ

Hiragana is for writing
Japanese words and is the first Japanese script.


Example: ひらがな (hiragana)


Show all the kana of Hiragana

Dakuten:

All the "s" kana change to "z" : ざ, じ, ず, ぜ, ぞ

All the "t" kana change to "d" : だ, ぢ, づ, で, ど

All the "h" kana change to "b" : ば, び, ぶ, べ, ぼ

All the "k" kana change to "g" : が, ぎ, ぐ, げ, ご

つ tsu changes to づ zu


Handakuten:

Also "h" changes to "p": ぱ, ぴ, ぷ, ぺ, ぽ

カ

Katakana is used for writing
foreign words and names and is the second Japanese script.


Example: カタカナ (katakana)


Show all the kana of Katakana

Dakuten:

All the "s" kana change to "z" : ザ, ジ, ズ, ゼ, ゾ

All the "t" kana change to "d" : ダ, ぢ, づ, で, ど

All the "h" kana change to "b" : バ, び, ぶ, べ, ぼ

All the "k" kana change to "g" : ガ, ぎ, ぐ, げ, ご

つ tsu changes to づ zu


Handakuten:

Also "h" changes to "p": パ, ピ, プ, ペ, ポ
